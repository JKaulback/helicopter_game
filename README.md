# Helicopter Game

A 2D side-scrolling helicopter game built with C++ and Raylib. Fly your helicopter, avoid obstacles, and shoot down incoming missiles!

## Controls

*   **Fly Up**: W / Up Arrow
*   **Move Left**: A / Left Arrow
*   **Move Right**: D / Right Arrow
*   **Shoot**: Space (Tap)
*   **Restart**: R (Game Over screen)

## Building the Project

This project uses CMake.

1.  Create a build directory:
    ```bash
    mkdir build
    cd build
    ```
2.  Generate build files:
    ```bash
    cmake ..
    ```
3.  Build the project:
    ```bash
    cmake --build .
    ```
4.  Run the game:
    ```bash
    ./HelicopterGame.exe
    ```
